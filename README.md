# MT 字幕翻译官

专业的 SRT 字幕翻译工具，支持 20 种语言，AI 驱动，一键批量翻译。

## 软件功能

- **SRT 字幕翻译**：导入 SRT 文件，选择目标语言，自动翻译并导出新字幕文件
- **标题翻译**：手动输入大段文字，一键翻译为多种语言，所有语言合并保存在同一个 txt 文件，顶部附原文对照，语言之间用星号分隔，Alt+Enter 快捷键触发翻译，点击文件名直接打开
- **封面生成**：将标题翻译结果一键渲染为多语言封面图片，支持横屏（YouTube）和竖屏（TikTok/抖音）尺寸
- **即时互译**：粘贴任意文字，实时翻译，支持自由切换源语言和目标语言，Ctrl+C 智能复制（有选中复制选中，无选中复制全文），Alt+D 一键清空
- **20 种语言支持**：简体中文、繁体中文、英语、日语、韩语、法语、德语、西班牙语等
- **多平台 API**：支持 DeepSeek、阿里云通义、OpenAI、Claude、Google Gemini 等主流翻译引擎
- **自定义接口地址**：API 设置页底部"高级设置"可填写中转/代理地址，按平台单独保存，留空则使用官方接口
- **AI 智能生成标题**：SRT 翻译完成后自动生成凝练版标题、标题党标题和内容摘要，也可在标题翻译页手动触发
- **激活码授权**：购买后通过私信收到激活码，一机一码，支持多年叠加续费
- **全局快捷键**：按 Alt+T 可随时将软件窗口呼出到最前台

## 下载安装

前往 [Releases](https://github.com/maxin8/mt/releases/latest) 页面下载 `mt.exe`，双击即可运行，无需安装。

## Windows 11 无法打开？

Windows 11 可能会拦截未经微软认证的软件，出现"Windows 已保护你的电脑"或文件被自动删除的情况。

**方法一：SmartScreen 弹窗（最常见）**

1. 弹窗出现后，点击**"更多信息"**
2. 再点击**"仍要运行"**即可正常启动

**方法二：文件属性解锁**

1. 右键 `mt.exe` → **属性**
2. 在窗口底部勾选 **"解除锁定"**（或"取消阻止"）
3. 点击**确定**，再次双击运行

**方法三：杀毒软件误报**

- 将 `mt.exe` 添加到杀毒软件的**白名单 / 信任列表**，或暂时关闭实时保护后运行

**方法四：智能应用控制（无法用方法一/二时）**

1. 打开 **Windows 安全中心** → **应用和浏览器控制**
2. 点击 **"智能应用控制设置"** → 选择 **"关闭"**
3. 重新双击软件即可正常运行

> 说明：软件不含任何恶意代码，SmartScreen 拦截是因为 exe 尚未获得微软代码签名证书，属于独立开发者软件的常见现象。

## 购买激活

访问爱发电页面购买激活码，付款后系统自动通过私信发送激活码。

打开软件 → 点击"激活"标签 → 输入激活码 → 点击立即激活。

激活码绑定机器，同一台电脑重装系统后可免费重新激活；有效期 1 年，续费从原到期日顺延。
