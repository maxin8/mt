# MT 字幕翻译官

专为视频创作者打造的 SRT 字幕 & 文字翻译工具，支持 20 种语言，AI 驱动，一键批量翻译。

## 主要功能

### SRT 字幕翻译
导入 SRT 文件，勾选目标语言（可多选），点击开始翻译，自动生成对应语言的字幕文件保存到指定目录。支持 20 种语言批量同时翻译。

### 标题翻译
输入视频标题或大段文字，一键翻译为多种语言，所有语言合并保存为一个 txt 文件（顶部附原文对照，语言间用星号分隔），Alt+Enter 快捷键触发翻译，点击文件名直接打开。

### 封面生成
根据「标题翻译」的结果，自动为每种语言渲染一张封面图，支持横屏（YouTube 1280×720）和竖屏（TikTok/抖音 1080×1920）。可调整标题位置、字号、文字颜色、背景色带和描边颜色，也可调用 AI API 优化文案后再渲染。

### 即时互译
粘贴或输入任意文字，边输入边翻译，实时显示结果。支持自由切换源语言和目标语言，按 ⇄ 键互换并重新翻译。Ctrl+C 智能复制（有选中复制选中，无选中复制全文），Alt+D 一键清空。

### 多平台 API 支持
支持 DeepSeek、阿里云通义千问、OpenAI GPT、Claude（Anthropic）、Google Gemini 五大翻译引擎，在「API 设置」页填写对应平台的 API Key 即可切换使用。点击任意平台行即自动切换为当前平台，无需额外操作。高级设置支持为每个平台单独配置中转代理地址。

### AI 智能生成标题
SRT 翻译完成后自动生成凝练版标题、标题党标题和内容摘要，填入「标题翻译」输入框，可直接翻译保存，也可手动修改后再用。支持设置标题风格（简中 / 适中 / 详细）和摘要长度。

### 激活码授权系统
- 免费版：支持简体中文、繁体中文、英语翻译
- 满血版：解锁全部 20 种语言，含日语、韩语、法语、德语、西班牙语、阿拉伯语等
- 激活码一机一码，同一台电脑重装系统后可免费重新激活
- 有效期 1 年，续费从原到期日顺延（支持提前续费叠加）

## Windows 11 无法打开 / 被拦截怎么办？

Windows 11 会对未经微软代码签名的独立开发者软件进行拦截，属于正常的安全机制，软件本身不含任何恶意代码。根据遇到的情况，按以下方法处理：

### 情况一：弹出"Windows 已保护你的电脑"（最常见）

1. 点击弹窗中的 **"更多信息"**
2. 点击 **"仍要运行"**，即可正常启动

### 情况二：文件被标记为"已锁定"

1. 右键 `mt.exe` → **属性**
2. 在窗口底部找到 **"解除锁定"** 或 **"取消阻止"** 复选框，打勾
3. 点击 **确定**，再次双击运行

### 情况三：杀毒软件自动删除或隔离

- 打开杀毒软件（360、火绒、Windows Defender 等）→ 找到隔离区 → 恢复文件
- 将 `mt.exe` 所在文件夹添加到**白名单 / 信任区**，防止再次被删除

### 情况四：以上方法均无效（智能应用控制开启）

1. 打开 **Windows 安全中心**（任务栏盾牌图标）
2. 进入 **应用和浏览器控制** → **智能应用控制设置**
3. 将模式改为 **"关闭"**
4. 重新双击 `mt.exe` 运行

> 若关闭智能应用控制后仍无法运行，可尝试右键 `mt.exe` → **以管理员身份运行**。
